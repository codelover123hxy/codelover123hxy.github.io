大二学年&技术部时光总结

发表于 2023-09-18

大二的收获

转眼间，大二学年已经过去，我也从精弘技术部卸任。回顾起这段时光，有许多事情值得记录。
我完成了当时许下的愿望，保持绩点综测双第一，绩点突破 4.2。
同时仍有许多遗憾，在服创、机器人大赛中失败，这会令我认识到比赛没有想象中那么容易。

技术上的突破

我完成了许多前端项目

表白墙试用期项目

这个项目是我进入精弘后完成的第一个项目，效果不尽人意，但让我意识到差距。其他大一技术部成员有的做得非常不错。

可视化项目

这个项目是我第一次接触 vue + 脚手架开发。我选用了 vue3+vite，通过网上教程学会了路由配置和自定义组件。我通过 echarts 引入图表可视化。

寒假软件设计项目——综测系统

这是我第一次作为组长领导项目开发。我独自完成整个前端部分，并且与后端相对接。项目做得还不错，但仍有遗憾。这是我第二次用微信小程序原生开发。

医智搜小程序

第一次使用 uniapp 技术栈完成开发，使用组件库完成渲染。

禾小联小程序

互联网+项目。继续使用 uniapp，uviews 组件库、ucharts 图标组件。相比之前，开发效率有了极大的突破。半小时可以完成一个页面。

朝夕之居 app

另一个互联网+项目 app。用 uniapp 开发打包为安卓小程序。明年继续完善。

数据库课设

第一次独立完成前后端开发。在暑假后期完成部署阿里云。以前都是做后端的朋友帮忙部署，现在发现也不难。

爱因社网站

半天时间搭建雏形。目前还在完善中。

未来的开发任务

非遗小程序

打算采用 taro + react 开发。
